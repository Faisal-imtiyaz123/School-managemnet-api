## Environment Setup

To run the project locally, you need to configure the following environment variables:

1. **Create a `.env` file** in the root of the project. You will need to give values for the following env variables:

```env
DB_HOST
DB_PORT
DB_USER
DB_PASSWORD
DB_NAME
PORT
NODE_ENV
```
2. Example .env file:
```env
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=""
DB_NAME=mysql
PORT=3000
NODE_ENV='production'
